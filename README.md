# meteor-build-deploy
Automation docker container to build and deploy with Meteor Up (mup).

## Programs
- node
- meteor
- git
- sudo
- curl
- python
- build-essential
- ca-certificates
- locales
- unicode-data

## User
- builder

## Extra features:
 * Can upload pictures to AWS S3 using `s3-upload` which is added to the PATH.
